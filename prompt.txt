- Styling: Apply a clean, modern theme to the UI.

Please design a practical, well-architected solution with the following requirements:

- Clean and minimal user interface
- Live Markdown preview (side-by-side or toggle mode)
- Syntax highlighting for Markdown
- Support for opening, editing, and saving .md files
- Keyboard shortcuts (save, open, bold, italic, etc.)
